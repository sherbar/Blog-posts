How to Connect With SSH

On a Mac, open the Terminal. You will be on your device, and we want to connect to the server. We need to connect to the server by typing ssh username@64.227.19.180. Type 'clear' to have a clear screen. Set up directory inside the server by typing $ cd/../../var/www/html. By doing this, you will be going up two levels and into the html. 'Cd home' to find your username. Use 'ls -al' to see a full list of folders in the directory. Make a directory by typing 'sudo mkdir 'username'' and enter the password. Ls -al to see the directory. Change the owner of the directory by typing sudo chown username and rename the file. Cd the file name. Mkdir scaffold. Ls -al to see all the folders, scafold folder should now be in the file.
